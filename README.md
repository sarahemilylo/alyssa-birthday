## 📝 Description

This is a personalized birthday interactive card website for **Alyssa**. The website features a series of interactive birthday-themed experiences, including animations, a birthday card, a candle-blowing celebration, and a photo booth gallery showcasing memorable moments.

The website guides visitors through several stages:

- ✨ Welcome screen
- 💌 Interactive envelope opening
- 🎂 Birthday card and candle-blowing celebration
- 🎉 Confetti animation
- 📸 Photo booth memory gallery

## ▶️ How to Run

### Run Locally

1. Clone the repository:

```bash
git clone https://github.com/yourusername/happy-birthday-alyssa.git
```

2. Open:

```text
index.html
```

in any modern web browser.

### GitHub Pages

1. Upload the project files to a GitHub repository.
2. Navigate to:

```text
Settings → Pages
```

3. Select:

```text
Deploy from a branch
```

4. Choose:

```text
main → /root
```

5. Save and wait for deployment.
